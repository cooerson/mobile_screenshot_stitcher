# 手机截图智能拼接工具使用说明

## 功能特点

🎯 **智能拼接算法**
- 优先检测头尾完全相同的部分
- 自动检测图片重合部分
- 去头尾后拼接重合区域
- 保留原有头尾内容
- 支持多张图片连续拼接

📱 **手机截图优化**
- 专门针对手机截图设计
- 自动识别截图特征
- 智能处理不同尺寸图片
- 保持图片质量

⚡ **高效处理**
- 实时预览和进度显示
- 可调节相似度阈值
- 支持批量处理
- 快速下载结果

## 使用方法

### 1. 上传图片
- **点击上传**：点击上传区域选择图片文件
- **拖拽上传**：直接将图片文件拖拽到上传区域
- **支持格式**：JPG、PNG、GIF等常见图片格式
- **数量要求**：至少需要2张图片才能进行拼接

### 2. 图片管理
- **预览图片**：上传后可以预览每张图片
- **调整顺序**：使用"上移"、"下移"按钮调整拼接顺序
- **删除图片**：点击"删除"按钮移除不需要的图片
- **自动排序**：点击"自动排序"按文件名数字顺序排列

### 3. 参数设置

#### 相似度阈值 (60% - 95%)
- **低阈值 (60-70%)**：更宽松的匹配，适合内容差异较大的图片
- **中等阈值 (75-85%)**：平衡的匹配，适合大多数情况
- **高阈值 (90-95%)**：严格的匹配，只拼接高度相似的区域

#### 检测范围 (10% - 40%)
- **小范围 (10-20%)**：只在图片边缘很小区域检测重合
- **中等范围 (20-30%)**：在图片边缘较大区域检测重合
- **大范围 (30-40%)**：在图片边缘很大区域检测重合

#### 输出质量
- **高质量 (90%)**：文件较大，质量最好
- **标准质量 (80%)**：平衡质量和文件大小
- **压缩质量 (60%)**：文件较小，质量一般

### 4. 开始拼接
1. 确保已上传至少2张图片
2. 调整参数设置（可选）
3. 点击"开始智能拼接"按钮
4. 等待处理完成

**拼接优先级**：
- 优先检测头尾完全相同的部分
- 其次检测普通重叠区域
- 最后直接拼接（无重合）

### 5. 查看结果
- **拼接统计**：显示最终尺寸、拼接次数、处理时间、头尾检测次数、重叠检测次数
- **预览图片**：查看拼接后的完整图片
- **下载图片**：点击"下载图片"保存到本地
- **复制到剪贴板**：直接复制图片到剪贴板

## 拼接原理

### 1. 图片预处理
- 加载所有上传的图片
- 按高度排序确定拼接顺序
- 分析图片尺寸和特征

### 2. 头尾检测
- 优先检测图片头部和尾部完全相同的部分
- 最多检测30%的头部和尾部区域
- 确保头部和尾部都匹配才进行拼接

### 3. 重合检测
- 在图片边缘区域检测相同内容
- 使用像素相似度算法比较
- 找到最佳重合位置和高度

### 4. 智能拼接
- 优先去除头尾相同部分
- 其次去除普通重合部分
- 按顺序拼接所有图片
- 保持图片完整性和连续性

### 5. 结果生成
- 生成最终拼接图片
- 应用质量设置
- 提供下载和分享功能

### 常见问题

**Q: 拼接效果不理想怎么办？**
A: 尝试调整相似度阈值和检测范围，确保图片顺序正确

**Q: 处理速度太慢？**
A: 减少图片数量或降低输出质量，使用压缩质量选项

**Q: 重合检测不准确？**
A: 提高相似度阈值，确保图片有足够的重合内容

**Q: 头尾检测有什么优势？**
A: 头尾检测能更精确地处理手机截图，避免重复内容，生成更干净的长图

**Q: 为什么需要头尾都相同？**
A: 确保拼接的连续性和完整性，避免内容断裂或重复

**Q: 支持的图片格式？**
A: 支持JPG、PNG、GIF、WebP等常见格式

**Q: 最大支持多少张图片？**
A: 理论上无限制，但建议不超过20张以保证性能
